# Streams API Bootcamp

A collection of Java Streams API problems and solutions, implemented for practice and learning.  
This project demonstrates how to use functional programming concepts in Java with Streams, including filtering, mapping, reducing, and more.

## 🚀 Features
- Examples of common **Streams operations** (filter, map, reduce, collect).
- Practice problems covering **real-world use cases**.
- Clean, simple solutions with focus on readability.
- Organized project structure for easy navigation.

## 📂 Project Structure
- `src/main/java` → Contains all Java source files.
- Each problem is implemented as a separate class for clarity.

## 🛠️ Requirements
- Java 17 or higher
- Maven/Gradle (optional, for build automation)
